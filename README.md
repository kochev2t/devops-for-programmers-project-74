# Project: Docker compose app

This is a simple training app for learning docker compose main functions and basic ci principles

## Requirements

* docker
* docker compose 
* make

## Usage

### Setup app

```bash
make setup
```

### Run tests

```bash
make test
```

### Start application

```bash
make dev
# open http://0.0.0.0:8080 in browser
```
